# 麻雀点数計算アプリ

大会用の麻雀点数計算Webアプリケーションです。オーラス時に各プレイヤーが目標順位を達成するために必要な最小点数を計算します。

## 特徴

- 📱 **スマートフォン対応**: レスポンシブデザインでモバイル最適化
- 🔌 **オフライン動作**: PWA対応でネットワーク不要
- 🎯 **3つのルール対応**:
  - 日本プロ麻雀連盟公式ルール（12点加減方式）
  - WRCルール
  - WRC-Rルール
- 🧮 **詳細な条件計算**:
  - 各プレイヤーが1位になる条件
  - 3位→2位、4位→2位になる条件
  - ツモ・ロン別の最小点数表示

## 使い方

### 1. ルール選択

最初の画面で大会のルールを選択します：
- 日本プロ麻雀連盟公式ルール
- WRCルール
- WRC-Rルール

### 2. スコア入力

以下の情報を入力します：
- プレイヤー名（省略可、デフォルトは東家・南家・西家・北家）
- 3回戦終了時スコア（累計）
- オーラス現在スコア
- 親の位置
- 立直棒の数
- 積み棒の数

### 3. 結果確認

各プレイヤーについて以下が表示されます：
- 現在の順位と点数
- 1位になるための条件
  - ツモの場合の最小点数
  - 各プレイヤーからロンした場合の最小点数
- 3位→2位、4位→2位になる条件（該当する場合）

## ルール詳細

### 日本プロ麻雀連盟公式ルール

- 30000点持ち30000点返し
- 12点加減方式（浮き人数により変動）
- 同着の場合は順位点を折半
- 30符4翻は切り上げなし（子7700/親11600）

### WRCルール

- 30000点持ち30000点返し
- 順位点: 1着+15、2着+5、3着-5、4着-15
- 同点の場合は順位点を分割
- 30符4翻は切り上げ満貫（子8000/親12000）

### WRC-Rルール

- 30000点持ち30000点返し
- 順位点: 1着+30、2着+15、3着-15、4着-30
- 同点の場合は順位点を分割
- 30符4翻は切り上げ満貫（子8000/親12000）

## 対応点数

- 通常役: 70符まで対応
- 役満: 3倍役満まで対応
